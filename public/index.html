What is Houston Herald?

The Houston Herald is an application developed by a singular developer using HTML, CSS, and JS.
This application has the purpose of suggesting attractions to visitors to Houston without having to deal with suboptimally performing websites.


Frameworks


Express JS is a framework part of the "OpenJS Foundation" which is a foundation that pushes for the smooth growth of JavaScript.
Express is the go-to framework for web application development in Node.JS due to the easily understandable and well supported HTTP utilities.


Electron JS is another framework part of the the "OpenJS Foundation".
Electron is one of few options for building cross-platform desktop applications with web-based languages.
This allows developers to easily expand their reach with an application without having to spend major time reworking an application for specifing platforms.


Font Awesome is a CSS framework that allows developers to use SVGs on their websites with minimal hassle.


Libraries


better-sqlite3

Better-SQLITE3 is the fastest library for Sqlite3 (in JS) which is a database that does not require a server.
This allows Sqlite to be easily configured for small scale projects and promote better runtimes.


tablesort

Tablesort is a lightweight (558 kB) script that provides a simple syntax for sorting tables based on headers.
It also comes with a stylesheet which allows a developer to easily include a more modern look to their tables.
